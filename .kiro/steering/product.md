# Product Overview

## Productivity Dashboard

A lightweight, browser-based productivity application that provides essential tools in a single, clean interface. The dashboard helps users stay organized and focused with four core features:

- **Time-based Greeting**: Displays current time, date, and contextual greeting messages
- **Focus Timer**: 25-minute Pomodoro-style countdown timer for work sessions
- **To-Do List**: Task management with creation, editing, completion tracking, and filtering
- **Quick Links**: Bookmark manager for frequently visited websites

## Key Characteristics

- **Zero Dependencies**: Built with vanilla HTML, CSS, and JavaScript
- **Client-Side Only**: No backend required, runs entirely in the browser
- **Persistent Storage**: Uses Local Storage API for data persistence across sessions
- **Responsive Design**: Works on both desktop and mobile devices
- **Accessibility Focused**: Includes proper ARIA labels and keyboard navigation

## Target Users

Individuals seeking a simple, distraction-free productivity tool that doesn't require account creation, installation, or internet connectivity after initial load.
